# Laboratorio de configuración de servidor Linux

Este laboratorio muestra la instalación y configuración básica de un servidor Linux (Ubuntu/CentOS).

## Contenido:
- Creación de usuarios y permisos.
- Configuración de servicios esenciales.
- Instalación de herramientas de administración (top, htop, net-tools, vim, etc.).
- Configuración básica de firewall y seguridad.
- Documentación de los pasos para referencia futura.

Este laboratorio sirve como base para la administración de sistemas y preparación para otros laboratorios de bases de datos y monitoreo.
